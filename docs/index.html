Overview

This internal website for the Damrauer Lab at the University of Pennsylvania serves to coordinate and document common logistic and computational topics. This effort depends on contributions from users, so if you have edits to suggest, or documentation requests, please submit an Issue or Pull Request. This effort is inspired in part by PennLINC.
